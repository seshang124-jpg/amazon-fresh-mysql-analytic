# Amazon-fresh-mysql-analytic
 MySQL-powered analytics on Amazon Fresh data — covering DDL, DML, joins, subqueries, normalization &amp; business insights.
# About the Project
This project demonstrates end-to-end database analysis on a simulated Amazon Fresh dataset using MySQL Workbench. It covers database design, querying, and deriving actionable business insights across 14 structured tasks.
# Tasks Covered
ER Diagram & Table Relationships 
Primary Keys & Foreign Keys 
Basic SELECT Queries4DDL Statements & Constraints 
INSERT – Add New Products 
UPDATE – Modify Stock Quantity 
DELETE – Remove Supplier by City 
CHECK & DEFAULT Constraints 
WHERE, HAVING, GROUP BY, ORDER BY 
Top Customers by Total Spending 
JOIN Queries – Revenue & Suppliers 
Normalization to 3NF
Subqueries – Top Products & Inactive Customers
Actionable Business Insights
#  Key Insights
 Identified cities with the highest concentration of Prime members
 Ranked customers by total spending using DENSE_RANK()
 Found top 3 best-selling product categories
 Normalized the Products table to 3NF for better data integrity
 # Tech Stack
Database: MySQL
Tool: MySQL Workbench
Language: SQL
